# Linux Log Analyzer

A simple Bash script that analyzes Linux authentication logs and displays useful security information.

## Features

* Count failed login attempts
* Count successful logins
* Display the top 5 IP addresses
* Parse log files using `grep`, `sort`, `uniq`, `wc`, and `head`

## Technologies Used

* Bash Shell Scripting
* Linux (Ubuntu)
* Command-line utilities

## How to Run

```bash
chmod +x log_analyzer.sh
./log_analyzer.sh
```

## Sample Output

* Failed Password Attempts
* Successful Logins
* Top 5 IP Addresses

## Learning Objectives

This project demonstrates:

* Shell scripting fundamentals
* Linux log analysis
* Text processing with Bash
* File handling and command-line automation
